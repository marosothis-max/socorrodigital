# Logo de SocorroDigital

## Instrucciones para agregar tu logo

1. **Guarda tu imagen del logo** (la que muestra el diseño de Venezuela con personas migrando) en este directorio con el nombre:
   
   ```
   socorro-digital-logo.png
   ```

2. **Ubicación exacta**: 
   ```
   D:\Proyectos\SocorroDigital\public\images\logo\socorro-digital-logo.png
   ```

3. **Formato recomendado**:
   - Formato: PNG con fondo transparente
   - Tamaño recomendado: 500x500 píxeles o similar (cuadrado)
   - Resolución: 72-150 DPI

4. **Alternativas aceptadas**:
   - También puedes usar `.svg` (vector) cambiando la extensión en `components/ui/Logo.tsx`
   - O `.jpg` / `.webp` si no necesitas transparencia

## El logo aparecerá en:

- ✅ Header (arriba a la izquierda, al lado del nombre)
- ✅ Hero section (grande en el centro, animado)
- ✅ Iconos flotantes de fondo
- ✅ Footer (abajo a la izquierda)

## Después de agregar el logo:

1. Guarda la imagen en esta carpeta
2. Reinicia el servidor de desarrollo si está corriendo:
   ```bash
   # Detén el servidor (Ctrl + C)
   # Vuelve a iniciarlo
   npm run dev
   ```
3. Refresca el navegador (Ctrl + Shift + R)
4. ¡Tu logo debería aparecer!

## Si el logo no aparece:

- Verifica que el nombre del archivo sea exactamente: `socorro-digital-logo.png`
- Verifica que esté en la carpeta correcta
- Verifica que la imagen no esté corrupta
- Revisa la consola del navegador por errores
