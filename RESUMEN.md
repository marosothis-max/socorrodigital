## 🚀 Próximos pasos para ti

### 1. Subir a GitHub
```bash
# Crea un nuevo repositorio en github.com llamado "socorrodigital"
# Luego ejecuta:
git remote add origin https://github.com/TU_USUARIO/socorrodigital.git
git branch -M main
git push -u origin main
```

### 2. Desplegar en Vercel
1. Ve a [vercel.com](https://vercel.com)
2. Conecta con GitHub
3. Importa el repositorio `socorrodigital`
4. Deploy automático

Tu sitio estará en: `https://socorrodigital.vercel.app`

### 3. Personalizar
- Actualiza el email en `config/site.ts`
- Agrega tu usuario de GitHub en el README
- Configura canales de contacto (WhatsApp/Telegram)

### 4. Expandir contenido
- Agrega más tutoriales en `content/espana-migracion.ts`
- Crea archivos para otras categorías
- Agrega contenido para Francia y EEUU
